git init >инизиализация репозитория
git status проверка состояния файлов
git add выбор файлов для коммита
git commit сохраняем текущий слепок проекта
git log просмотр всех коммитов с их идентификаторами
git checkout позволяет переключаться между ветками или коммитами
git branch показывает существующие ветки
git push загружает файлы которые в коммите в глобальный репозиторий
git pull считываем файлы из глобального репозитория
git merge объединяем разные ветки
git clone делает копию репозитория

.gitignore скрывает от гита указанные файлы/папки

git revert позволяет отменить коммиты
git reset позволяет удалить коммит и его историю из ветки
git rebase объединяет ветки коппируя наборы коммитов из одной в другую
git cherry-pick копирует прописанные коммиты в head
git stash временно сохраняет все незафиксированные изменения отслеживаемых файлов
